EnableReturnKey

Enable you navigate through controls with Return key as with tab key

Syntax:

Function EnableReturnKey(ByVal dm_Form As System.Windows.Forms.Form, ByVal Mode As Boolean)

new in version3.5

Now DC_ReturnKeyEnable can work within UserControl Control

Function EnableReturnKey(ByVal dm_Form As System.Windows.Forms.UserControl, ByVal Mode As Boolean)

 

Example:

Dim RKE As New DynamicComponents.ReturnKeyEnable()

RKE.EnableReturnKey(True)
