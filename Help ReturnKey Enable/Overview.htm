Overview

DC ReturnKeyEnable
DC ReturnKeyEnable allows you to access form controls using the Return key instead of Tab. This way, you don’t have to switch between keys to enter codes, access controls, and navigate through your form. With DC ReturnKeyEnable, you can spend less time defining events and more time perfecting your application.

Simple, concise programming
It only takes a single line of code to enable the Return key – no need for additional programming and repetitive codes. DC ReturnKeyEnable enters the code at the beginning of your form, instead of defining each event and re-entering the code throughout your project.

Works with all COM-based programs
DC ReturnKeyEnable can be loaded onto any COM-based application, including Borland C++, Visual C++, Visual Basic, and Delphi. It also works with major programming languages used with these applications, so you won’t have to learn a whole new script to enable the Return key.

Automatic installation and setup
DC ReturnKeyEnable registers the key files automatically on Windows 98, NT, 2000, XP, Windows 7 & 8 operating systems.
